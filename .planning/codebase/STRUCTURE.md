# Structure

## Directory Layout
The project is currently transitioning from a Create React App structure to a Next.js App Router structure under `nextjs_rebase/`.

- `nextjs_rebase/src/app/`: Next.js routing (layout.tsx, page.tsx).
- `nextjs_rebase/src/components/`: Reusable React user interface components (Header, Footer, Projects).
- `nextjs_rebase/src/contexts/`: React Context providers (ThemeContext).
- `nextjs_rebase/src/hooks/`: Custom React hooks (useGitHub, useTheme).
- `nextjs_rebase/src/services/`: API and business logic services.
- `nextjs_rebase/src/styles/`: Global styles and styled-components themes.
- `nextjs_rebase/src/types/`: TypeScript interface definitions.
